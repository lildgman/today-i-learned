# 구현
- **풀이를 떠올리는거는 쉽지만 소스코드로 옮기기 어려운 문제**
- 예시
  - 간단한 알고리즘이지만 코드가 길어지는 문제
  - 실수 연산을 다루고, 특정 소수점 자리까지 출력해야 하는 문제
  - 문자열을 특정한 기준에 따라 끊어 처리해야 하는 문제
  - 적절한 라이브러리를 찾아 사용해야 하는 문제

일반적으로 알고리즘 문제에서는 2차원 공간은 **행렬**의 의미로 사용된다.
